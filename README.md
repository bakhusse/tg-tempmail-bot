# tg-tempmail-bot
🚀 Твой личный почтовый сервис в Telegram на собственных доменах. Забудь про бан-листы временных почт. 100% проходимость на любых сайтах (Discord, Steam, Social Club).

📌 Основные фишки
Свои домены: Используй любые домены, купленные на Porkbun или других регистраторах.

Real-time Notifications: Уведомления о новых письмах прилетают мгновенно без опроса API (используется SSE).

Full Webmail: Кнопка входа в полноценный веб-интерфейс для отправки писем и работы с вложениями.

Custom Names: Возможность создать ящик с любым префиксом или рандомным логином.

Privacy: Каждый ящик жестко привязан к Telegram ID пользователя.

🛠 Технологии
Python 3.12

Aiogram 3.x (Framework для бота)

Httpx / Aiohttp (Асинхронные запросы)

SMTP.dev API (Почтовый шлюз)

SQLite (База данных пользователей)

🚀 Быстрый старт
Склонируй репо.

Создай venv и установи зависимости: pip install -r requirements.txt.

Настрой config.py (токен бота и API ключ smtp.dev).

Пропиши MX-запись mx.smtp.dev в настройках своего домена.

Запускай: python main.py.
